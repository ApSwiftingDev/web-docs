4. ¿QUÉ TECNOLOGÍAS USA LA APP?

Control de cuenta, hash técnico y protección antifraude

La app utiliza un sistema de control técnico que permite identificar de forma segura a los usuarios sin necesidad de conocer su identidad real. Al iniciar sesión con Apple, se crea una cuenta que sirve para verificar el acceso autorizado a los servicios disponibles. Además, se genera automáticamente un identificador técnico (hash) que se usa para gestionar funciones como las suscripciones premium o el uso de servicios externos. Este identificador no puede ser revertido ni vinculado a tu identidad personal.

En condiciones normales, el usuario podrá eliminar su cuenta por completo desde los ajustes de la aplicación, lo que implicará la eliminación tanto de su cuenta de Firebase como del identificador técnico (hash) vinculado a esta. En caso de uso fraudulento, el acceso a la app puede ser bloqueado y esa cuenta se inhabilita. Si esto ocurre, la cuenta no podrá ser eliminada por el usuario y se conservará de forma permanente como medida de seguridad para garantizar la integridad del sistema. Esto implica que el usuario no podrá volver a iniciar sesión en la app con ese mismo Apple ID, ya que dicho identificador quedará restringido para futuros accesos. Esta decisión es definitiva y forma parte de las medidas de protección adoptadas por el desarrollador frente a usos indebidos o maliciosos.

Todo esto se hace exclusivamente para proteger el servicio, evitar abusos y garantizar que cada usuario acceda solo a lo que le corresponde. No se usan estos datos para fines comerciales, de perfilado o publicidad.
